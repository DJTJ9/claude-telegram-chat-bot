# Claude Config Sync Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Turn `~/.claude` into a private GitHub repo that auto-syncs across Windows devices on every session start/stop.

**Architecture:** `~/.claude` gets `git init` + private GitHub remote. Existing `config-pull.sh` / `config-push.sh` hooks already handle pull-on-start and push-on-stop — they just need a remote. ECC plugin moves from local `file` path to `github` source. `skills/watch` is gitignored and re-cloned by `setup.ps1`. Memory dirs are preserved via targeted gitignore patterns.

**Tech Stack:** Git, PowerShell, GitHub (private repo), Claude Code settings.json

### Task 5: Create `setup.ps1`

**Files:**
- Create: `C:\Users\tjark\.claude\setup.ps1`

- [ ] **Step 1: Create the bootstrap script**

Create `C:\Users\tjark\.claude\setup.ps1`:

```powershell
# Claude Code Setup — run once on a new Windows device
# Prerequisites: git, Node.js, Python 3 must already be installed
# Claude Code must be installed separately: https://claude.ai/download

param(
    [switch]$SkipProjects
)

$ErrorActionPreference = "Stop"

Write-Host "=== Claude Code Setup ===" -ForegroundColor Cyan

# 1. Node deps for hooks
Write-Host "`n[1/4] Installing hook dependencies..."
npm install --prefix "$HOME\.claude\hooks" --silent
Write-Host "OK"

# 2. Clone projects (unless skipped)
if (-not $SkipProjects) {
    Write-Host "`n[2/4] Cloning projects..."

    New-Item -ItemType Directory -Force "C:\Projekte" | Out-Null

    $telegramBotPath = "C:\Projekte\telegram-notion-bot"
    if (-not (Test-Path $telegramBotPath)) {
        git clone https://github.com/DJTJ9/telegram-notion-bot $telegramBotPath
        Write-Host "  Cloned telegram-notion-bot"
    } else {
        Write-Host "  telegram-notion-bot already exists, skipping"
    }

    # skills/watch has its own .git — excluded from dotfiles repo, re-cloned here
    $watchPath = "$HOME\.claude\skills\watch"
    if (-not (Test-Path $watchPath)) {
        git clone https://github.com/bradautomates/claude-video.git $watchPath
        Write-Host "  Cloned skills/watch"
    } else {
        Write-Host "  skills/watch already exists, skipping"
    }
} else {
    Write-Host "`n[2/4] Skipping project clones (-SkipProjects)"
}

# 3. Python deps
Write-Host "`n[3/4] Installing Python dependencies..."
$reqFile = "C:\Projekte\telegram-notion-bot\requirements.txt"
if (Test-Path $reqFile) {
    pip install -r $reqFile --quiet
    Write-Host "OK"
} else {
    Write-Host "  Skipped (telegram-notion-bot not found at expected path)"
}

# 4. Manual steps reminder
Write-Host "`n[4/4] Manual steps required:" -ForegroundColor Yellow
Write-Host "  1. Open Claude Code and enter your API key"
Write-Host "  2. Create C:\Projekte\telegram-notion-bot\.env with:"
Write-Host "       TELEGRAM_BOT_TOKEN=<your token>"
Write-Host "       NOTION_API_KEY=<your key>"
Write-Host "       NOTION_PAGE_ID=<your page id>"
Write-Host ""
Write-Host "  Note: If laptop username differs from 'tjark', update teach-local path in:"
Write-Host "        settings.json > extraKnownMarketplaces > teach-local > source > path"
Write-Host ""
Write-Host "=== Setup complete. Start Claude Code to load plugins. ===" -ForegroundColor Green
```
